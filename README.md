# Clothing Store - Hindu Clothes <img src="./public/logo.png" alt="Logo" width="75" style="vertical-align: middle; line-height: 1.5;">

This is a sleek and modern e-commerce SPA web application focused on traditional Indian clothing. It allows users to browse clothing collections, view item details, filter products, and manage their shopping experience in a smooth, user-friendly interface.

## Main Features ‚ú®

- **Product Catalog** ‚Äì Browse a variety of traditional Indian clothing for men and women
- **Product Detail Pages** ‚Äì View detailed descriptions, ratings, related products , sizes, images, and availability of selected product
- **Filtering & Sorting** ‚Äì Filter products by type, gender, price, color, availability, size and sort by price or popularity
- **Orders Functionality** ‚Äì Add, update, and browse your order history, pay your orders - (without payment gate)
- **Shopping Cart Functionality** ‚Äì Add, update, and remove items from the cart
- **Profile Functionality** ‚Äì Manage your profile data like email, password, avatar - (OFF for safety purposes - CORS)
- **Reusable & Modular Components** ‚Äì Cleanly destructured components for product cards, category selectors, etc.
- **State Management with ngRx** ‚Äì Stores communicate with backend, manage auth, product, cart, and user data  and track loading statuses independently
- **Responsive Design** ‚Äì Fully optimized for desktop, tablet, and mobile devices
- **Environment** ‚Äì .env files for environment variables

This project is architected for scalability, maintainability, and performance, following best practices in Angular, state management, and modular design. It serves as a practical example of building a modern, real-world e-commerce application with clean code structure, reusable components, and efficient data handling.
